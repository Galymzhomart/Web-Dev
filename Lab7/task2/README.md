# Lab 7 - Task 2 OOP Project

## Project Description
This project demonstrates Object-Oriented Programming in Python.

It includes:
- a base class `Animal`
- two child classes `Dog` and `Cat`

## OOP Concepts Used
- classes and objects
- inheritance
- method overriding
- polymorphism
- modules and imports

## Files
- `models.py` - contains all classes
- `main.py` - creates objects and demonstrates their usage

## How to Run
Run the following command:

```bash
python main.py
